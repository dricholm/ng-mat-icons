Ng Material Icons

A library for using Material Icons inside your Angular project.

Setup

  1. Install the library in your Angular project using npm: npm install ng-mat-icons
  2. Import the NgMatIconsModule in your module. The recommended way is to include it in a SharedModule, which also exports it. Then import the SharedModule in other modules of your application.

  3. Now you can begin to use Material Icons in your components.

Usage

The library contains one component called NgMatIcons. This can be used with the <nmi-icon> HTML tag. You can specify several attributes to customize the icon. By default the icon will inherit the size and color from the parent element.

Basic usage

The content between the HTML tags defines which icon to display. It can be given using the icon name or the code point. You can find the full list of icons on the Material Design website, while the list of code points can be found in the material-design-icons GitHub repo.

Accessibility and label

By default the icon will be hidden for screen readers by using the aria-hidden="true" attribute. Using the label attribute you can specify what gets displayed for screen readers by the aria-label attribute. Setting this attribute will set aria-hidden to false.

Size

The icon will inherit the font size from the parent element. You can specify the size attribute to use the recommended sizes specified in the Material Icons Guide.

The size can be given in pixels (18, 24, 36 or 48) or using sm, md, lg or xl.

Color

The icon inherits the color from the parent element by default. This can be overriden to use the colors specified in the Material Icons Guide by the color attribute. If the color is set to light or dark then the inactive property can also be set to true. The inactive attribute only works if the color attribute is set.
